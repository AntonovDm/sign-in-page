# Sign in page.

Работа выполнена для y_lab_university. 

Демо можно посмотреть здесь → [Demo](https://sign-in-page-ylab.vercel.app/)

## Stack

- React
- TypeScript
- SCSS module
- React Router
- Небольшой адаптив для веб и мобильных устройств

### Getting started

```bash
npm install
npm run dev
```
