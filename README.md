# Sign in page.

Работа выполнена для y_lab_university с адаптивом для веб и мобильных устройств.

Демо можно посмотреть здесь → [Demo](https://sign-in-page-ylab.vercel.app/)

## Stack

- React
- TypeScript
- SCSS module
- React Router
- Reack Hook Form
- Yup

### Getting started

```bash
npm install
npm run dev
```
